# Васюкинский шахматный турнир - пример верстки
Вёрстка лендинга для Международного васюкинского шахматного турнира.

---

## Технологии

- HTML5 — семантическая разметка
- SCSS — препроцессор, архитектура по методологии БЭМ
- Vanilla JS — без библиотек и фреймворков
- Gulp — сборка проекта (компиляция SCSS, минификация, live reload)
- gulp-file-include — инклюды HTML партиалов
- BrowserSync — локальный сервер с hot reload

---

## Структура проекта

```
src/                        # Исходники
  assets/                   # Изображения и медиафайлы
  html/
    components/             # Переиспользуемые HTML компоненты
    modules/                # Секции страницы
    svg/                    # SVG иконки
  js/
    components/             # JS компоненты (карусели, тикер)
    utils/                  # Утилиты (debounce, throttle)
    main.js                 # Точка входа
  scss/
    base/                   # Сброс стилей, переменные, типографика
    components/             # UI компоненты (кнопки, карточки)
    layout/                 # Стили секций страницы
    main.scss               # Точка входа

dist/                       # Скомпилированная версия для продакшна
  assets/                   # Изображения
  css/main.css              # Скомпилированный и минифицированный CSS
  js/main.js                # Скомпилированный и минифицированный JS
  index.html                # Готовая страница
```

---

## Запуск

### Установка зависимостей
```bash
npm install
```

### Режим разработки
```bash
npm run dev
```
Откроется локальный сервер `http://localhost:3000` с hot reload.

### Сборка продакшн версии
```bash
npm run build
```

### Просмотр без сборки
Открыть `dist/index.html` в браузере — сборка уже включена в репозиторий.

## Реализованный функционал

- Адаптивная вёрстка для экранов 375px и 1366px
- Промежуточные брейкпоинты добавлены самостоятельно
- Бегущая строка — CSS анимация с динамическим клонированием через JS
- Карусель участников — бесконечная, автопрокрутка каждые 4 секунды
- Карусель этапов — конечная, без автопрокрутки, управление кнопками
- Якорные кнопки на hero секции

---

## Примечания
JS компоненты реализованы через паттерн IIFE Module Pattern
для изоляции области видимости без использования ES modules.
